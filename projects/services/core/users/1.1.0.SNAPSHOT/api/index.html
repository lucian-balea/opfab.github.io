TimeLineFormats - Up

Timeline rendered date / time formats
dateInsideTooltipsWeek (optional)
String Default format use on date inside tooltips for time's unit lower than hour
dateInsideTooltipsMonth (optional)
String Default format use on date inside tooltips for other case
dateOnDay (optional)
String Default format use for display a day and month informations with day name, number, and month
dateOnWeek (optional)
String Default format use for display a week information with day number, month number and two last year's number
dateOnMonth (optional)
String Default format use for display a month information with month name and two last year's number
dateOnYear (optional)
String Default format use for display a year information
titleDateInsideTooltips (optional)
String Default format use on first part of title inside tooltips showing day number and month number of its date
titleHourInsideTooltips (optional)
String Default format use on second part of title inside tooltips showing hours and minutes of its date
dateOnDayNewYear (optional)
String Default format use for display first year's day on a day configuration display day number, month name and two last year's number
realTimeBarFormat (optional)
String Default format use on real time bar date, show minutes, hours, day number, month number and two last year's number
dateSimplifliedOnDayNewYear (optional)
String Default format use on date configuration for first year's day, show simplified day number, month name and last two year's number
dateSimplifliedOnDay (optional)
String Default format use on date configuration, show simplified day number and month name
hoursOnly (optional)
String Default format for show only hours
minutesOnly (optional)
String Default format for show only minutes
secondedsOnly (optional)
String Default format for show only seconds
weekNumberOnly (optional)
String Default format for show only week number

User associated settings. Note that the current supported locales are en and fr. Date and time formats use Moment.js formats.

String User login
email (optional)
String User email
description (optional)
String Free user description label (ex: organization role)
timeZone (optional)
String User time zone (See https://momentjs.com)
locale (optional)
String User using browser format
timeFormat (optional)
String Time formatting (See https://momentjs.com)
dateFormat (optional)
String Date formatting (See https://momentjs.com)
dateTimeFormat (optional)
String Date Time formatting (See https://momentjs.com). If not set dateFormat and timeFormat are used to deduce date time format.
