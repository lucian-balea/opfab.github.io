EpochDate - Up

Number of milliseconds since Epoch (long integer)

SpeedEnum - Up

Speed of time flow in virtual time >

  • X1 - Normal speed
  • X2 - Twice as fast
  • X10 - Ten times as fast
  • X60 - 60 times as fast (1s = 1min)
  • X3600 - 3600 times as fast (1s = 1hour)
  • HALF - Half as fast

TimeData - Up

Description of a time configuration, containing the current virtual time as well as the references used for computation
referenceTime (optional)
EpochDate Real server time at last time change
virtualTime (optional)
EpochDate Chosen virtual time start at last time change
computedNow (optional)
EpochDate Current virtual time computed by server as virtualTime + (now - referenceTime) * speed
speed (optional)
SpeedEnum Speed of virtual time flow
