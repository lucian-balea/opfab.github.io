Up 
get /process/{processInstanceId}/states/{state}/actions/{actionKey}
Get action associated to process with updated status (getAction)
Get action associated with Third+process+state, returns an array of actions (application/json)

Path parameters

processInstanceId (required)
Path Parameter — Process instance unique id
state (required)
Path Parameter — Name of state
actionKey (required)
Path Parameter — key of action (unique for state)

Return type

map[ActionStatus]

Example data

Content-Type: application/json
{
  "key" : {
    "type" : "URL",
    "lockAction" : true,
    "lockCard" : true,

    "inputs" : [ {
      "type" : "LIST",
      "label" : {

      "mandatory" : true,
      "name" : "selectedColour",
      "value" : {
        "value" : "#000000",
        "label" : {
          "key" : "black",

            "EN" : "Paint It Black",

          }
        }
      },
      "values" : [ {

ActionEnum Action type
url (optional)

lockCard (optional)
Boolean if true, all card action will be locked after clicked client-side
needsConfirm (optional)
Boolean if true, a confirmation will be asked before execution client-side

Boolean True if action has already been called at least once

Boolean If true, client will attempt an update of ActionStatus before rendering
updateStateBeforeAction (optional)
Boolean if true, client will attempt an update of ActionStatus before execution

Boolean This action is hidden from card and is only available to details;
contentStyle (optional)
String css classes of action button content (a <i> tag)

I18n button default label, appears as a tooltip

array[Input] list of action input parameters, generate a popup form upon action call

Input - Up

Describe an input parameter for an Action
type
InputEnum Parameter type, may imply some mandatory attributes, see description
label
I18n internationalized label
mandatory (optional)
Boolean Is the parameter required
name
String Parameter name (unique in action scope)
value (optional)
String Parameter default value
maxLength (optional)
Integer Only for TEXT and LONGTEXT parameters. Specifies the maximum allowed character count
rows (optional)
Integer LONGTEXT parameter. Specifies the maximum number of displayed rows
values (optional)
array[ParameterListItem] Only for LIST,MULTI_LIST and SWITCH_LIST parameters. Specifies the selectable items in a list
selectedValues (optional)
array[String] Only for MULTI_LIST and SWITCH_LIST parameters. List selected value. Order matters.
unSelectedValues (optional)
array[String] Only for SWITCH_LIST parameters. List unselected values. Order matters.

InputEnum - Up

Input type >

  • TEXT - A simple text input (size not limited)
  • LONGTEXT - A long text input (size not limited)
  • BOOLEAN - A boolean input
  • LIST - A list with one selectable value
  • MULTI_LIST - A list with multiple selectable values
  • SWITCH_LIST - Two lists with exchangeable values
  • STATIC - A static non editable parameter

ParameterListItem - Up

An item of list input parameter
value
String parameter technical value
label (optional)
I18n Value i18n label
